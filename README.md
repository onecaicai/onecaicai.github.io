## ⚡ 快速开始

### 本地预览
```bash
# 启动开发服务器
hugo server --buildDrafts

# 访问 http://localhost:1313
```

### 创建文章
```bash
# 新建文章
hugo new posts/文章标题.md

# 编辑文章内容，然后本地预览
```

### 部署到线上
```bash
# 一键部署
./deploy.sh
```
